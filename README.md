# GA
Genetic algorithm for image segmentation

* otsu_2D : 二维otsu算法
* otsu_threshold: 多阈值分割
大标题  
===================================

中标题  
-----------------------------------  
  中标题一般显示重点项,类似html的\<h2\><br/>
